# 금현물 잔고확인 (kt50020)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 계좌 > 금현물 잔고확인(kt50020)
- **API 명**: 금현물 잔고확인
- **API ID**: kt50020
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/acnt`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **tot_entr** (String, 12, 선택): 예수금
- **net_entr** (String, 12, 선택): 추정예수금
- **tot_est_amt** (String, 12, 선택): 잔고평가액
- **net_amt** (String, 12, 선택): 예탁자산평가액
- **tot_book_amt2** (String, 12, 선택): 총매입금액
- **tot_dep_amt** (String, 12, 선택): 추정예탁자산
- **paym_alowa** (String, 12, 선택): 출금가능금액
- **pl_amt** (String, 12, 선택): 실현손익
- **gold_acnt_evlt_prst** (LIST, , 선택): 금현물계좌평가현황
  - **stk_cd** (String, 30, 선택): 종목코드
  - **stk_nm** (String, 12, 선택): 종목명
  - **real_qty** (String, 12, 선택): 보유수량
  - **avg_prc** (String, 12, 선택): 평균단가
  - **cur_prc** (String, 12, 선택): 현재가
  - **est_amt** (String, 12, 선택): 평가금액
  - **est_lspft** (String, 12, 선택): 손익금액
  - **est_ratio** (String, 12, 선택): 손익율
  - **cmsn** (String, 12, 선택): 수수료
  - **vlad_tax** (String, 12, 선택): 부가가치세
  - **book_amt2** (String, 12, 선택): 매입금액
  - **pl_prch_prc** (String, 12, 선택): 손익분기매입가
  - **qty** (String, 12, 선택): 결제잔고
  - **buy_qty** (String, 12, 선택): 매수수량
  - **sell_qty** (String, 12, 선택): 매도수량
  - **able_qty** (String, 12, 선택): 가능수량
