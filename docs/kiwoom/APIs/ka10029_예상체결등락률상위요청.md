# 예상체결등락률상위요청 (ka10029)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 순위정보 > 예상체결등락률상위요청(ka10029)
- **API 명**: 예상체결등락률상위요청
- **API ID**: ka10029
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/rkinfo`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **mrkt_tp** (String, 3, 필수): 시장구분 (000:전체, 001:코스피, 101:코스닥)
- **sort_tp** (String, 1, 필수): 정렬구분 (1:상승률, 2:상승폭, 3:보합, 4:하락률, 5:하락폭, 6:체결량, 7:상한, 8:하한)
- **trde_qty_cnd** (String, 5, 필수): 거래량조건 (0:전체조회, 1;천주이상, 3:3천주, 5:5천주, 10:만주이상, 50:5만주이상, 100:10만주이상)
- **stk_cnd** (String, 2, 필수): 종목조건 (0:전체조회, 1:관리종목제외, 3:우선주제외, 4:관리종목,우선주제외, 5:증100제외, 6:증100만보기, 7:증40만보기, 8:증30만보기, 9:증20만보기, 11:정리매매종목제외, 12:증50만보기, 13:증60만보기, 14:ETF제외, 15:스팩제외, 16:ETF+ETN제외)
- **crd_cnd** (String, 1, 필수): 신용조건 (0:전체조회, 1:신용융자A군, 2:신용융자B군, 3:신용융자C군, 4:신용융자D군, 5:신용한도초과제외, 7:신용융자E군, 8:신용대주, 9:신용융자전체)
- **pric_cnd** (String, 2, 필수): 가격조건 (0:전체조회, 1:1천원미만, 2:1천원~2천원, 3:2천원~5천원, 4:5천원~1만원, 5:1만원이상, 8:1천원이상, 10:1만원미만)
- **stex_tp** (String, 1, 필수): 거래소구분 (1:KRX, 2:NXT 3.통합)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **exp_cntr_flu_rt_upper** (LIST, , 선택): 예상체결등락률상위
  - **stk_cd** (String, 20, 선택): 종목코드
  - **stk_nm** (String, 40, 선택): 종목명
  - **exp_cntr_pric** (String, 20, 선택): 예상체결가
  - **base_pric** (String, 20, 선택): 기준가
  - **pred_pre_sig** (String, 20, 선택): 전일대비기호
  - **pred_pre** (String, 20, 선택): 전일대비
  - **flu_rt** (String, 20, 선택): 등락률
  - **exp_cntr_qty** (String, 20, 선택): 예상체결량
  - **sel_req** (String, 20, 선택): 매도잔량
  - **sel_bid** (String, 20, 선택): 매도호가
  - **buy_bid** (String, 20, 선택): 매수호가
  - **buy_req** (String, 20, 선택): 매수잔량
