# 종목별기관매매추이요청 (ka10045)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 시세 > 종목별기관매매추이요청(ka10045)
- **API 명**: 종목별기관매매추이요청
- **API ID**: ka10045
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/mrkcond`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **stk_cd** (String, 20, 필수): 종목코드 (거래소별 종목코드
(KRX:039490,NXT:039490_NX,SOR:039490_AL))
- **strt_dt** (String, 8, 필수): 시작일자 (YYYYMMDD)
- **end_dt** (String, 8, 필수): 종료일자 (YYYYMMDD)
- **orgn_prsm_unp_tp** (String, 1, 필수): 기관추정단가구분 (1:매수단가, 2:매도단가)
- **for_prsm_unp_tp** (String, 1, 필수): 외인추정단가구분 (1:매수단가, 2:매도단가)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **orgn_prsm_avg_pric** (String, , 선택): 기관추정평균가
- **for_prsm_avg_pric** (String, , 선택): 외인추정평균가
- **stk_orgn_trde_trnsn** (LIST, , 선택): 종목별기관매매추이
  - **dt** (String, 20, 선택): 일자
  - **close_pric** (String, 20, 선택): 종가
  - **pre_sig** (String, 20, 선택): 대비기호
  - **pred_pre** (String, 20, 선택): 전일대비
  - **flu_rt** (String, 20, 선택): 등락율
  - **trde_qty** (String, 20, 선택): 거래량
  - **orgn_dt_acc** (String, 20, 선택): 기관기간누적
  - **orgn_daly_nettrde_qty** (String, 20, 선택): 기관일별순매매수량
  - **for_dt_acc** (String, 20, 선택): 외인기간누적
  - **for_daly_nettrde_qty** (String, 20, 선택): 외인일별순매매수량
  - **limit_exh_rt** (String, 20, 선택): 한도소진율
