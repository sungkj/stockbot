# 시가대비등락률요청 (ka10028)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 종목정보 > 시가대비등락률요청(ka10028)
- **API 명**: 시가대비등락률요청
- **API ID**: ka10028
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/stkinfo`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **sort_tp** (String, 1, 필수): 정렬구분 (1:시가, 2:고가, 3:저가, 4:기준가)
- **trde_qty_cnd** (String, 4, 필수): 거래량조건 (0000:전체조회, 0010:만주이상, 0050:5만주이상, 0100:10만주이상, 0500:50만주이상, 1000:백만주이상)
- **mrkt_tp** (String, 3, 필수): 시장구분 (000:전체, 001:코스피, 101:코스닥)
- **updown_incls** (String, 1, 필수): 상하한포함 (0:불 포함, 1:포함)
- **stk_cnd** (String, 2, 필수): 종목조건 (0:전체조회, 1:관리종목제외, 4:우선주+관리주제외, 3:우선주제외, 5:증100제외, 6:증100만보기, 7:증40만보기, 8:증30만보기, 9:증20만보기)
- **crd_cnd** (String, 1, 필수): 신용조건 (0:전체조회, 1:신용융자A군, 2:신용융자B군, 3:신용융자C군, 4:신용융자D군, 7:신용융자E군, 9:신용융자전체)
- **trde_prica_cnd** (String, 4, 필수): 거래대금조건 (0:전체조회, 3:3천만원이상, 5:5천만원이상, 10:1억원이상, 30:3억원이상, 50:5억원이상, 100:10억원이상, 300:30억원이상, 500:50억원이상, 1000:100억원이상, 3000:300억원이상, 5000:500억원이상)
- **flu_cnd** (String, 1, 필수): 등락조건 (1:상위, 2:하위)
- **stex_tp** (String, 1, 필수): 거래소구분 (1:KRX, 2:NXT 3.통합)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **open_pric_pre_flu_rt** (LIST, , 선택): 시가대비등락률
  - **stk_cd** (String, 20, 선택): 종목코드
  - **stk_nm** (String, 40, 선택): 종목명
  - **cur_prc** (String, 20, 선택): 현재가
  - **pred_pre_sig** (String, 20, 선택): 전일대비기호
  - **pred_pre** (String, 20, 선택): 전일대비
  - **flu_rt** (String, 20, 선택): 등락률
  - **open_pric** (String, 20, 선택): 시가
  - **high_pric** (String, 20, 선택): 고가
  - **low_pric** (String, 20, 선택): 저가
  - **open_pric_pre** (String, 20, 선택): 시가대비
  - **now_trde_qty** (String, 20, 선택): 현재거래량
  - **cntr_str** (String, 20, 선택): 체결강도
