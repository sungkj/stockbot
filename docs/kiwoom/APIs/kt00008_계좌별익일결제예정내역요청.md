# 계좌별익일결제예정내역요청 (kt00008)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 계좌 > 계좌별익일결제예정내역요청(kt00008)
- **API 명**: 계좌별익일결제예정내역요청
- **API ID**: kt00008
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/acnt`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **strt_dcd_seq** (String, 7, 선택): 시작결제번호

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **trde_dt** (String, 8, 선택): 매매일자
- **setl_dt** (String, 8, 선택): 결제일자
- **sell_amt_sum** (String, 12, 선택): 매도정산합
- **buy_amt_sum** (String, 12, 선택): 매수정산합
- **acnt_nxdy_setl_frcs_prps_array** (LIST, , 선택): 계좌별익일결제예정내역배열
  - **seq** (String, 7, 선택): 일련번호
  - **stk_cd** (String, 12, 선택): 종목번호
  - **loan_dt** (String, 8, 선택): 대출일
  - **qty** (String, 12, 선택): 수량
  - **engg_amt** (String, 12, 선택): 약정금액
  - **cmsn** (String, 12, 선택): 수수료
  - **incm_tax** (String, 12, 선택): 소득세
  - **rstx** (String, 12, 선택): 농특세
  - **stk_nm** (String, 40, 선택): 종목명
  - **sell_tp** (String, 10, 선택): 매도수구분
  - **unp** (String, 12, 선택): 단가
  - **exct_amt** (String, 12, 선택): 정산금액
  - **trde_tax** (String, 12, 선택): 거래세
  - **resi_tax** (String, 12, 선택): 주민세
  - **crd_tp** (String, 20, 선택): 신용구분
