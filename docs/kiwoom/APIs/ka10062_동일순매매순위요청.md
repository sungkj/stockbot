# 동일순매매순위요청 (ka10062)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 순위정보 > 동일순매매순위요청(ka10062)
- **API 명**: 동일순매매순위요청
- **API ID**: ka10062
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/rkinfo`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **strt_dt** (String, 8, 필수): 시작일자 (YYYYMMDD
(연도4자리, 월 2자리, 일 2자리 형식))
- **end_dt** (String, 8, 선택): 종료일자 (YYYYMMDD
(연도4자리, 월 2자리, 일 2자리 형식))
- **mrkt_tp** (String, 3, 필수): 시장구분 (000:전체, 001: 코스피, 101:코스닥)
- **trde_tp** (String, 1, 필수): 매매구분 (1:순매수, 2:순매도)
- **sort_cnd** (String, 1, 필수): 정렬조건 (1:수량, 2:금액)
- **unit_tp** (String, 1, 필수): 단위구분 (1:단주, 1000:천주)
- **stex_tp** (String, 1, 필수): 거래소구분 (1:KRX, 2:NXT 3.통합)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **eql_nettrde_rank** (LIST, , 선택): 동일순매매순위
  - **stk_cd** (String, 20, 선택): 종목코드
  - **rank** (String, 20, 선택): 순위
  - **stk_nm** (String, 40, 선택): 종목명
  - **cur_prc** (String, 20, 선택): 현재가
  - **pre_sig** (String, 20, 선택): 대비기호
  - **pred_pre** (String, 20, 선택): 전일대비
  - **flu_rt** (String, 20, 선택): 등락율
  - **acc_trde_qty** (String, 20, 선택): 누적거래량
  - **orgn_nettrde_qty** (String, 20, 선택): 기관순매매수량
  - **orgn_nettrde_amt** (String, 20, 선택): 기관순매매금액
  - **orgn_nettrde_avg_pric** (String, 20, 선택): 기관순매매평균가
  - **for_nettrde_qty** (String, 20, 선택): 외인순매매수량
  - **for_nettrde_amt** (String, 20, 선택): 외인순매매금액
  - **for_nettrde_avg_pric** (String, 20, 선택): 외인순매매평균가
  - **nettrde_qty** (String, 20, 선택): 순매매수량
  - **nettrde_amt** (String, 20, 선택): 순매매금액
