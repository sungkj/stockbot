# 프로그램매매추이요청 일자별 (ka90010)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 시세 > 프로그램매매추이요청 일자별(ka90010)
- **API 명**: 프로그램매매추이요청 일자별
- **API ID**: ka90010
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/mrkcond`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **date** (String, 8, 필수): 날짜 (YYYYMMDD)
- **amt_qty_tp** (String, 1, 필수): 금액수량구분 (1:금액(백만원), 2:수량(천주))
- **mrkt_tp** (String, 10, 필수): 시장구분 (코스피- 거래소구분값 1일경우:P00101, 2일경우:P001_NX01, 3일경우:P001_AL01
코스닥- 거래소구분값 1일경우:P10102, 2일경우:P101_NX02, 3일경우:P001_AL02)
- **min_tic_tp** (String, 1, 필수): 분틱구분 (0:틱, 1:분)
- **stex_tp** (String, 1, 필수): 거래소구분 (1:KRX, 2:NXT 3.통합)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **prm_trde_trnsn** (LIST, , 선택): 프로그램매매추이
  - **cntr_tm** (String, 20, 선택): 체결시간
  - **dfrt_trde_sel** (String, 20, 선택): 차익거래매도
  - **dfrt_trde_buy** (String, 20, 선택): 차익거래매수
  - **dfrt_trde_netprps** (String, 20, 선택): 차익거래순매수
  - **ndiffpro_trde_sel** (String, 20, 선택): 비차익거래매도
  - **ndiffpro_trde_buy** (String, 20, 선택): 비차익거래매수
  - **ndiffpro_trde_netprps** (String, 20, 선택): 비차익거래순매수
  - **dfrt_trde_sell_qty** (String, 20, 선택): 차익거래매도수량
  - **dfrt_trde_buy_qty** (String, 20, 선택): 차익거래매수수량
  - **dfrt_trde_netprps_qty** (String, 20, 선택): 차익거래순매수수량
  - **ndiffpro_trde_sell_qty** (String, 20, 선택): 비차익거래매도수량
  - **ndiffpro_trde_buy_qty** (String, 20, 선택): 비차익거래매수수량
  - **ndiffpro_trde_netprps_qty** (String, 20, 선택): 비차익거래순매수수량
  - **all_sel** (String, 20, 선택): 전체매도
  - **all_buy** (String, 20, 선택): 전체매수
  - **all_netprps** (String, 20, 선택): 전체순매수
  - **kospi200** (String, 20, 선택): KOSPI200
  - **basis** (String, 20, 선택): BASIS
