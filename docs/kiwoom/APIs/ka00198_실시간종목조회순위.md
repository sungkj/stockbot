# 실시간종목조회순위 (ka00198)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 종목정보 > 실시간종목조회순위(ka00198)
- **API 명**: 실시간종목조회순위
- **API ID**: ka00198
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/stkinfo`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **qry_tp** (String, 1, 필수): 구분 (1:1분, 2:10분, 3:1시간, 4:당일 누적, 5:30초)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **item_inq_rank** (LIST, , 선택): 실시간종목조회순위
  - **stk_nm** (String, 40, 선택): 종목명
  - **bigd_rank** (String, 20, 선택): 빅데이터 순위
  - **rank_chg** (String, 20, 선택): 순위 등락
  - **rank_chg_sign** (String, 20, 선택): 순위 등락 부호
  - **past_curr_prc** (String, 20, 선택): 과거 현재가
  - **base_comp_sign** (String, 20, 선택): 기준가 대비 부호
  - **base_comp_chgr** (String, 20, 선택): 기준가 대비 등락율
  - **prev_base_sign** (String, 20, 선택): 직전 기준 대비 부호
  - **prev_base_chgr** (String, 20, 선택): 직전 기준 대비 등락율
  - **dt** (String, 20, 선택): 일자
  - **tm** (String, 20, 선택): 시간
  - **stk_cd** (String, 20, 선택): 종목코드
