# 거래원순간거래량요청 (ka10052)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 종목정보 > 거래원순간거래량요청(ka10052)
- **API 명**: 거래원순간거래량요청
- **API ID**: ka10052
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/stkinfo`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **mmcm_cd** (String, 3, 필수): 회원사코드 (회원사 코드는 ka10102 조회)
- **stk_cd** (String, 20, 선택): 종목코드 (거래소별 종목코드
(KRX:039490,NXT:039490_NX,SOR:039490_AL))
- **mrkt_tp** (String, 1, 필수): 시장구분 (0:전체, 1:코스피, 2:코스닥, 3:종목)
- **qty_tp** (String, 3, 필수): 수량구분 (0:전체, 1:1000주, 2:2000주, 3:, 5:, 10:10000주, 30: 30000주, 50: 50000주, 100: 100000주)
- **pric_tp** (String, 1, 필수): 가격구분 (0:전체, 1:1천원 미만, 8:1천원 이상, 2:1천원 ~ 2천원, 3:2천원 ~ 5천원, 4:5천원 ~ 1만원, 5:1만원 이상)
- **stex_tp** (String, 1, 필수): 거래소구분 (1:KRX, 2:NXT 3.통합)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **trde_ori_mont_trde_qty** (LIST, , 선택): 거래원순간거래량
  - **tm** (String, 20, 선택): 시간
  - **stk_cd** (String, 20, 선택): 종목코드
  - **stk_nm** (String, 20, 선택): 종목명
  - **trde_ori_nm** (String, 20, 선택): 거래원명
  - **tp** (String, 20, 선택): 구분
  - **mont_trde_qty** (String, 20, 선택): 순간거래량
  - **acc_netprps** (String, 20, 선택): 누적순매수
  - **cur_prc** (String, 20, 선택): 현재가
  - **pred_pre_sig** (String, 20, 선택): 전일대비기호
  - **pred_pre** (String, 20, 선택): 전일대비
  - **flu_rt** (String, 20, 선택): 등락율
