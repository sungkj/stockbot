# 계좌평가현황요청 (kt00004)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 계좌 > 계좌평가현황요청(kt00004)
- **API 명**: 계좌평가현황요청
- **API ID**: kt00004
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/acnt`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **qry_tp** (String, 1, 필수): 상장폐지조회구분 (0:전체, 1:상장폐지종목제외)
- **dmst_stex_tp** (String, 6, 필수): 국내거래소구분 (KRX:한국거래소,NXT:넥스트트레이드)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **acnt_nm** (String, 30, 선택): 계좌명
- **brch_nm** (String, 30, 선택): 지점명
- **entr** (String, 12, 선택): 예수금
- **d2_entra** (String, 12, 선택): D+2추정예수금
- **tot_est_amt** (String, 12, 선택): 유가잔고평가액
- **aset_evlt_amt** (String, 12, 선택): 예탁자산평가액
- **tot_pur_amt** (String, 12, 선택): 총매입금액
- **prsm_dpst_aset_amt** (String, 12, 선택): 추정예탁자산
- **tot_grnt_sella** (String, 12, 선택): 매도담보대출금
- **tdy_lspft_amt** (String, 12, 선택): 당일투자원금
- **invt_bsamt** (String, 12, 선택): 당월투자원금
- **lspft_amt** (String, 12, 선택): 누적투자원금
- **tdy_lspft** (String, 12, 선택): 당일투자손익
- **lspft2** (String, 12, 선택): 당월투자손익
- **lspft** (String, 12, 선택): 누적투자손익
- **tdy_lspft_rt** (String, 12, 선택): 당일손익율
- **lspft_ratio** (String, 12, 선택): 당월손익율
- **lspft_rt** (String, 12, 선택): 누적손익율
- **stk_acnt_evlt_prst** (LIST, , 선택): 종목별계좌평가현황
  - **stk_cd** (String, 12, 선택): 종목코드
  - **stk_nm** (String, 30, 선택): 종목명
  - **rmnd_qty** (String, 12, 선택): 보유수량
  - **avg_prc** (String, 12, 선택): 평균단가
  - **cur_prc** (String, 12, 선택): 현재가
  - **evlt_amt** (String, 12, 선택): 평가금액
  - **pl_amt** (String, 12, 선택): 손익금액
  - **pl_rt** (String, 12, 선택): 손익율
  - **loan_dt** (String, 10, 선택): 대출일
  - **pur_amt** (String, 12, 선택): 매입금액
  - **setl_remn** (String, 12, 선택): 결제잔고
  - **pred_buyq** (String, 12, 선택): 전일매수수량
  - **pred_sellq** (String, 12, 선택): 전일매도수량
  - **tdy_buyq** (String, 12, 선택): 금일매수수량
  - **tdy_sellq** (String, 12, 선택): 금일매도수량
