# 위탁종합거래내역요청 (kt00015)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 계좌 > 위탁종합거래내역요청(kt00015)
- **API 명**: 위탁종합거래내역요청
- **API ID**: kt00015
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/acnt`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **strt_dt** (String, 8, 필수): 시작일자
- **end_dt** (String, 8, 필수): 종료일자
- **tp** (String, 1, 필수): 구분 (0:전체,1:입출금,2:입출고,3:매매,4:매수,5:매도,6:입금,7:출금,A:예탁담보대출입금,B:매도담보대출입금,C:현금상환(융자,담보상환),F:환전,M:입출금+환전,G:외화매수,H:외화매도,I:환전정산입금,J:환전정산출금)
- **stk_cd** (String, 12, 선택): 종목코드
- **crnc_cd** (String, 3, 선택): 통화코드
- **gds_tp** (String, 1, 필수): 상품구분 (0:전체, 1:국내주식, 2:수익증권, 3:해외주식, 4:금융상품)
- **frgn_stex_code** (String, 10, 선택): 해외거래소코드
- **dmst_stex_tp** (String, 6, 필수): 국내거래소구분 (%:(전체),KRX:한국거래소,NXT:넥스트트레이드)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **trst_ovrl_trde_prps_array** (LIST, , 선택): 위탁종합거래내역배열
  - **trde_dt** (String, 8, 선택): 거래일자
  - **trde_no** (String, 9, 선택): 거래번호
  - **rmrk_nm** (String, 60, 선택): 적요명
  - **crd_deal_tp_nm** (String, 20, 선택): 신용거래구분명
  - **exct_amt** (String, 15, 선택): 정산금액
  - **loan_amt_rpya** (String, 15, 선택): 대출금상환
  - **fc_trde_amt** (String, 15, 선택): 거래금액(외)
  - **fc_exct_amt** (String, 15, 선택): 정산금액(외)
  - **entra_remn** (String, 15, 선택): 예수금잔고
  - **crnc_cd** (String, 3, 선택): 통화코드
  - **trde_ocr_tp** (String, 2, 선택): 거래종류구분 (1:입출금, 2:펀드, 3:ELS, 4:채권, 5:해외채권, 6:외화RP, 7:외화발행어음)
  - **trde_kind_nm** (String, 20, 선택): 거래종류명
  - **stk_nm** (String, 40, 선택): 종목명
  - **trde_amt** (String, 15, 선택): 거래금액
  - **trde_agri_tax** (String, 15, 선택): 거래및농특세
  - **rpy_diffa** (String, 15, 선택): 상환차금
  - **fc_trde_tax** (String, 15, 선택): 거래세(외)
  - **dly_sum** (String, 15, 선택): 연체합
  - **fc_entra** (String, 15, 선택): 외화예수금잔고
  - **mdia_tp_nm** (String, 20, 선택): 매체구분명
  - **io_tp** (String, 1, 선택): 입출구분
  - **io_tp_nm** (String, 10, 선택): 입출구분명
  - **orig_deal_no** (String, 9, 선택): 원거래번호
  - **stk_cd** (String, 12, 선택): 종목코드
  - **trde_qty_jwa_cnt** (String, 30, 선택): 거래수량/좌수
  - **cmsn** (String, 15, 선택): 수수료
  - **int_ls_usfe** (String, 15, 선택): 이자/대주이용
  - **fc_cmsn** (String, 15, 선택): 수수료(외)
  - **fc_dly_sum** (String, 15, 선택): 연체합(외)
  - **vlbl_nowrm** (String, 30, 선택): 유가금잔
  - **proc_tm** (String, 111, 선택): 처리시간
  - **isin_cd** (String, 12, 선택): ISIN코드
  - **stex_cd** (String, 10, 선택): 거래소코드
  - **stex_nm** (String, 20, 선택): 거래소명
  - **trde_unit** (String, 20, 선택): 거래단가/환율
  - **incm_resi_tax** (String, 15, 선택): 소득/주민세
  - **loan_dt** (String, 8, 선택): 대출일
  - **uncl_ocr** (String, 30, 선택): 미수(원/주)
  - **rpym_sum** (String, 30, 선택): 변제합
  - **cntr_dt** (String, 8, 선택): 체결일
  - **rcpy_no** (String, 20, 선택): 출납번호
  - **prcsr** (String, 20, 선택): 처리자
  - **proc_brch** (String, 20, 선택): 처리점
  - **trde_stle** (String, 40, 선택): 매매형태
  - **txon_base_pric** (String, 15, 선택): 과세기준가
  - **tax_sum_cmsn** (String, 15, 선택): 세금수수료합
  - **frgn_pay_txam** (String, 15, 선택): 외국납부세액(외)
  - **fc_uncl_ocr** (String, 15, 선택): 미수(외)
  - **rpym_sum_fr** (String, 30, 선택): 변제합(외)
  - **rcpmnyer** (String, 20, 선택): 입금자
  - **trde_prtc_tp** (String, 2, 선택): 거래내역구분
