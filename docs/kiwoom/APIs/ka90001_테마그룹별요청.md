# 테마그룹별요청 (ka90001)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 테마 > 테마그룹별요청(ka90001)
- **API 명**: 테마그룹별요청
- **API ID**: ka90001
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/thme`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **qry_tp** (String, 1, 필수): 검색구분 (0:전체검색, 1:테마검색, 2:종목검색)
- **stk_cd** (String, 6, 선택): 종목코드 (검색하려는 종목코드)
- **date_tp** (String, 2, 필수): 날짜구분 (n일전 (1일 ~ 99일 날짜입력))
- **thema_nm** (String, 50, 선택): 테마명 (검색하려는 테마명)
- **flu_pl_amt_tp** (String, 1, 필수): 등락수익구분 (1:상위기간수익률, 2:하위기간수익률, 3:상위등락률, 4:하위등락률)
- **stex_tp** (String, 1, 필수): 거래소구분 (1:KRX, 2:NXT 3.통합)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **thema_grp** (LIST, , 선택): 테마그룹별
  - **thema_grp_cd** (String, 20, 선택): 테마그룹코드
  - **thema_nm** (String, 20, 선택): 테마명
  - **stk_num** (String, 20, 선택): 종목수
  - **flu_sig** (String, 20, 선택): 등락기호
  - **flu_rt** (String, 20, 선택): 등락율
  - **rising_stk_num** (String, 20, 선택): 상승종목수
  - **fall_stk_num** (String, 20, 선택): 하락종목수
  - **dt_prft_rt** (String, 20, 선택): 기간수익률
  - **main_stk** (String, 20, 선택): 주요종목
