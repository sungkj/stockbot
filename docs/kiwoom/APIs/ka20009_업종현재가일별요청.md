# 업종현재가일별요청 (ka20009)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 업종 > 업종현재가일별요청(ka20009)
- **API 명**: 업종현재가일별요청
- **API ID**: ka20009
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/sect`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **mrkt_tp** (String, 1, 필수): 시장구분 (0:코스피, 1:코스닥, 2:코스피200)
- **inds_cd** (String, 3, 필수): 업종코드 (001:종합(KOSPI), 002:대형주, 003:중형주, 004:소형주 101:종합(KOSDAQ), 201:KOSPI200, 302:KOSTAR, 701: KRX100 나머지 ※ 업종코드 참고)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **cur_prc** (String, 20, 선택): 현재가
- **pred_pre_sig** (String, 20, 선택): 전일대비기호
- **pred_pre** (String, 20, 선택): 전일대비
- **flu_rt** (String, 20, 선택): 등락률
- **trde_qty** (String, 20, 선택): 거래량
- **trde_prica** (String, 20, 선택): 거래대금
- **trde_frmatn_stk_num** (String, 20, 선택): 거래형성종목수
- **trde_frmatn_rt** (String, 20, 선택): 거래형성비율
- **open_pric** (String, 20, 선택): 시가
- **high_pric** (String, 20, 선택): 고가
- **low_pric** (String, 20, 선택): 저가
- **upl** (String, 20, 선택): 상한
- **rising** (String, 20, 선택): 상승
- **stdns** (String, 20, 선택): 보합
- **fall** (String, 20, 선택): 하락
- **lst** (String, 20, 선택): 하한
- **52wk_hgst_pric** (String, 20, 선택): 52주최고가
- **52wk_hgst_pric_dt** (String, 20, 선택): 52주최고가일
- **52wk_hgst_pric_pre_rt** (String, 20, 선택): 52주최고가대비율
- **52wk_lwst_pric** (String, 20, 선택): 52주최저가
- **52wk_lwst_pric_dt** (String, 20, 선택): 52주최저가일
- **52wk_lwst_pric_pre_rt** (String, 20, 선택): 52주최저가대비율
- **inds_cur_prc_daly_rept** (LIST, , 선택): 업종현재가_일별반복
  - **dt_n** (String, 20, 선택): 일자n
  - **cur_prc_n** (String, 20, 선택): 현재가n
  - **pred_pre_sig_n** (String, 20, 선택): 전일대비기호n
  - **pred_pre_n** (String, 20, 선택): 전일대비n
  - **flu_rt_n** (String, 20, 선택): 등락률n
  - **acc_trde_qty_n** (String, 20, 선택): 누적거래량n
