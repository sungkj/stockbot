# 일별거래상세요청 (ka10015)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 종목정보 > 일별거래상세요청(ka10015)
- **API 명**: 일별거래상세요청
- **API ID**: ka10015
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/stkinfo`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **stk_cd** (String, 20, 필수): 종목코드 (거래소별 종목코드
(KRX:039490,NXT:039490_NX,SOR:039490_AL))
- **strt_dt** (String, 8, 필수): 시작일자 (YYYYMMDD)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **daly_trde_dtl** (LIST, , 선택): 일별거래상세
  - **dt** (String, 20, 선택): 일자
  - **close_pric** (String, 20, 선택): 종가
  - **pred_pre_sig** (String, 20, 선택): 전일대비기호
  - **pred_pre** (String, 20, 선택): 전일대비
  - **flu_rt** (String, 20, 선택): 등락율
  - **trde_qty** (String, 20, 선택): 거래량
  - **trde_prica** (String, 20, 선택): 거래대금
  - **bf_mkrt_trde_qty** (String, 20, 선택): 장전거래량
  - **bf_mkrt_trde_wght** (String, 20, 선택): 장전거래비중
  - **opmr_trde_qty** (String, 20, 선택): 장중거래량
  - **opmr_trde_wght** (String, 20, 선택): 장중거래비중
  - **af_mkrt_trde_qty** (String, 20, 선택): 장후거래량
  - **af_mkrt_trde_wght** (String, 20, 선택): 장후거래비중
  - **tot_3** (String, 20, 선택): 합계3
  - **prid_trde_qty** (String, 20, 선택): 기간중거래량
  - **cntr_str** (String, 20, 선택): 체결강도
  - **for_poss** (String, 20, 선택): 외인보유
  - **for_wght** (String, 20, 선택): 외인비중
  - **for_netprps** (String, 20, 선택): 외인순매수
  - **orgn_netprps** (String, 20, 선택): 기관순매수
  - **ind_netprps** (String, 20, 선택): 개인순매수
  - **frgn** (String, 20, 선택): 외국계
  - **crd_remn_rt** (String, 20, 선택): 신용잔고율
  - **prm** (String, 20, 선택): 프로그램
  - **bf_mkrt_trde_prica** (String, 20, 선택): 장전거래대금
  - **bf_mkrt_trde_prica_wght** (String, 20, 선택): 장전거래대금비중
  - **opmr_trde_prica** (String, 20, 선택): 장중거래대금
  - **opmr_trde_prica_wght** (String, 20, 선택): 장중거래대금비중
  - **af_mkrt_trde_prica** (String, 20, 선택): 장후거래대금
  - **af_mkrt_trde_prica_wght** (String, 20, 선택): 장후거래대금비중
