# 증권사별매매상위요청 (ka10039)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 순위정보 > 증권사별매매상위요청(ka10039)
- **API 명**: 증권사별매매상위요청
- **API ID**: ka10039
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/rkinfo`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **mmcm_cd** (String, 3, 필수): 회원사코드 (회원사 코드는 ka10102 조회)
- **trde_qty_tp** (String, 4, 필수): 거래량구분 (0:전체, 5:5000주, 10:1만주, 50:5만주, 100:10만주, 500:50만주, 1000: 100만주)
- **trde_tp** (String, 2, 필수): 매매구분 (1:순매수, 2:순매도)
- **dt** (String, 2, 필수): 기간 (1:전일, 5:5일, 10:10일, 60:60일)
- **stex_tp** (String, 1, 필수): 거래소구분 (1:KRX, 2:NXT 3.통합)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **sec_trde_upper** (LIST, , 선택): 증권사별매매상위
  - **rank** (String, 20, 선택): 순위
  - **stk_cd** (String, 20, 선택): 종목코드
  - **stk_nm** (String, 40, 선택): 종목명
  - **prid_stkpc_flu** (String, 20, 선택): 기간중주가등락
  - **flu_rt** (String, 20, 선택): 등락율
  - **prid_trde_qty** (String, 20, 선택): 기간중거래량
  - **netprps** (String, 20, 선택): 순매수
  - **buy_trde_qty** (String, 20, 선택): 매수거래량
  - **sel_trde_qty** (String, 20, 선택): 매도거래량
  - **netprps_amt** (String, 20, 선택): 순매수금액
  - **buy_amt** (String, 20, 선택): 매수금액
  - **sell_amt** (String, 20, 선택): 매도금액
