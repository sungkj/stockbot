# 업종별투자자순매수요청 (ka10051)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 업종 > 업종별투자자순매수요청(ka10051)
- **API 명**: 업종별투자자순매수요청
- **API ID**: ka10051
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/sect`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **mrkt_tp** (String, 1, 필수): 시장구분 (코스피:0, 코스닥:1)
- **amt_qty_tp** (String, 1, 필수): 금액수량구분 (금액:0, 수량:1)
- **base_dt** (String, 8, 선택): 기준일자 (YYYYMMDD)
- **stex_tp** (String, 1, 필수): 거래소구분 (1:KRX, 2:NXT, 3:통합)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **inds_netprps** (LIST, , 선택): 업종별순매수
  - **inds_cd** (String, 20, 선택): 업종코드
  - **inds_nm** (String, 20, 선택): 업종명
  - **cur_prc** (String, 20, 선택): 현재가
  - **pre_smbol** (String, 20, 선택): 대비부호
  - **pred_pre** (String, 20, 선택): 전일대비
  - **flu_rt** (String, 20, 선택): 등락율
  - **trde_qty** (String, 20, 선택): 거래량
  - **sc_netprps** (String, 20, 선택): 증권순매수
  - **insrnc_netprps** (String, 20, 선택): 보험순매수
  - **invtrt_netprps** (String, 20, 선택): 투신순매수
  - **bank_netprps** (String, 20, 선택): 은행순매수
  - **jnsinkm_netprps** (String, 20, 선택): 종신금순매수
  - **endw_netprps** (String, 20, 선택): 기금순매수
  - **etc_corp_netprps** (String, 20, 선택): 기타법인순매수
  - **ind_netprps** (String, 20, 선택): 개인순매수
  - **frgnr_netprps** (String, 20, 선택): 외국인순매수
  - **native_trmt_frgnr_netprps** (String, 20, 선택): 내국인대우외국인순매수
  - **natn_netprps** (String, 20, 선택): 국가순매수
  - **samo_fund_netprps** (String, 20, 선택): 사모펀드순매수
  - **orgn_netprps** (String, 20, 선택): 기관계순매수
