# 주식기본정보요청 (ka10001)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 종목정보 > 주식기본정보요청(ka10001)
- **API 명**: 주식기본정보요청
- **API ID**: ka10001
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/stkinfo`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **stk_cd** (String, 20, 필수): 종목코드 (거래소별 종목코드
(KRX:039490,NXT:039490_NX,SOR:039490_AL))

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **stk_cd** (String, 20, 선택): 종목코드
- **stk_nm** (String, 40, 선택): 종목명
- **setl_mm** (String, 20, 선택): 결산월
- **fav** (String, 20, 선택): 액면가
- **cap** (String, 20, 선택): 자본금
- **flo_stk** (String, 20, 선택): 상장주식
- **crd_rt** (String, 20, 선택): 신용비율
- **oyr_hgst** (String, 20, 선택): 연중최고
- **oyr_lwst** (String, 20, 선택): 연중최저
- **mac** (String, 20, 선택): 시가총액
- **mac_wght** (String, 20, 선택): 시가총액비중
- **for_exh_rt** (String, 20, 선택): 외인소진률
- **repl_pric** (String, 20, 선택): 대용가
- **per** (String, 20, 선택): PER ([ 주의 ] PER, ROE 값들은 외부벤더사에서 제공되는 데이터이며 일주일에 한번 또는 실적발표 시즌에 업데이트 됨)
- **eps** (String, 20, 선택): EPS
- **roe** (String, 20, 선택): ROE ([ 주의 ]  PER, ROE 값들은 외부벤더사에서 제공되는 데이터이며 일주일에 한번 또는 실적발표 시즌에 업데이트 됨)
- **pbr** (String, 20, 선택): PBR
- **ev** (String, 20, 선택): EV
- **bps** (String, 20, 선택): BPS
- **sale_amt** (String, 20, 선택): 매출액
- **bus_pro** (String, 20, 선택): 영업이익
- **cup_nga** (String, 20, 선택): 당기순이익
- **250hgst** (String, 20, 선택): 250최고
- **250lwst** (String, 20, 선택): 250최저
- **open_pric** (String, 20, 선택): 시가
- **high_pric** (String, 20, 선택): 고가
- **low_pric** (String, 20, 선택): 저가
- **upl_pric** (String, 20, 선택): 상한가
- **lst_pric** (String, 20, 선택): 하한가
- **base_pric** (String, 20, 선택): 기준가
- **exp_cntr_pric** (String, 20, 선택): 예상체결가
- **exp_cntr_qty** (String, 20, 선택): 예상체결수량
- **250hgst_pric_dt** (String, 20, 선택): 250최고가일
- **250hgst_pric_pre_rt** (String, 20, 선택): 250최고가대비율
- **250lwst_pric_dt** (String, 20, 선택): 250최저가일
- **250lwst_pric_pre_rt** (String, 20, 선택): 250최저가대비율
- **cur_prc** (String, 20, 선택): 현재가
- **pre_sig** (String, 20, 선택): 대비기호
- **pred_pre** (String, 20, 선택): 전일대비
- **flu_rt** (String, 20, 선택): 등락율
- **trde_qty** (String, 20, 선택): 거래량
- **trde_pre** (String, 20, 선택): 거래대비
- **fav_unit** (String, 20, 선택): 액면가단위
- **dstr_stk** (String, 20, 선택): 유통주식
- **dstr_rt** (String, 20, 선택): 유통비율
