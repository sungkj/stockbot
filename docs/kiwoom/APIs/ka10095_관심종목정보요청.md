# 관심종목정보요청 (ka10095)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 종목정보 > 관심종목정보요청(ka10095)
- **API 명**: 관심종목정보요청
- **API ID**: ka10095
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/stkinfo`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **stk_cd** (String, 20, 필수): 종목코드 (거래소별 종목코드
(KRX:039490,NXT:039490_NX,SOR:039490_AL)
여러개의 종목코드 입력시 | 로 구분)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **atn_stk_infr** (LIST, , 선택): 관심종목정보
  - **stk_cd** (String, 20, 선택): 종목코드
  - **stk_nm** (String, 40, 선택): 종목명
  - **cur_prc** (String, 20, 선택): 현재가
  - **base_pric** (String, 20, 선택): 기준가
  - **pred_pre** (String, 20, 선택): 전일대비
  - **pred_pre_sig** (String, 20, 선택): 전일대비기호
  - **flu_rt** (String, 20, 선택): 등락율
  - **trde_qty** (String, 20, 선택): 거래량
  - **trde_prica** (String, 20, 선택): 거래대금
  - **cntr_qty** (String, 20, 선택): 체결량
  - **cntr_str** (String, 20, 선택): 체결강도
  - **pred_trde_qty_pre** (String, 20, 선택): 전일거래량대비
  - **sel_bid** (String, 20, 선택): 매도호가
  - **buy_bid** (String, 20, 선택): 매수호가
  - **sel_1th_bid** (String, 20, 선택): 매도1차호가
  - **sel_2th_bid** (String, 20, 선택): 매도2차호가
  - **sel_3th_bid** (String, 20, 선택): 매도3차호가
  - **sel_4th_bid** (String, 20, 선택): 매도4차호가
  - **sel_5th_bid** (String, 20, 선택): 매도5차호가
  - **buy_1th_bid** (String, 20, 선택): 매수1차호가
  - **buy_2th_bid** (String, 20, 선택): 매수2차호가
  - **buy_3th_bid** (String, 20, 선택): 매수3차호가
