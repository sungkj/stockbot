# ETF시간대별추이요청 (ka40006)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > ETF > ETF시간대별추이요청(ka40006)
- **API 명**: ETF시간대별추이요청
- **API ID**: ka40006
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/etf`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **stk_cd** (String, 6, 필수): 종목코드

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **stk_nm** (String, 40, 선택): 종목명
- **etfobjt_idex_nm** (String, 20, 선택): ETF대상지수명
- **wonju_pric** (String, 20, 선택): 원주가격
- **etftxon_type** (String, 20, 선택): ETF과세유형
- **etntxon_type** (String, 20, 선택): ETN과세유형
- **etftisl_trnsn** (LIST, , 선택): ETF시간대별추이
  - **tm** (String, 20, 선택): 시간
  - **close_pric** (String, 20, 선택): 종가
  - **pre_sig** (String, 20, 선택): 대비기호
  - **pred_pre** (String, 20, 선택): 전일대비
  - **flu_rt** (String, 20, 선택): 등락율
  - **trde_qty** (String, 20, 선택): 거래량
  - **nav** (String, 20, 선택): NAV
  - **trde_prica** (String, 20, 선택): 거래대금
  - **navidex** (String, 20, 선택): NAV지수
  - **navetf** (String, 20, 선택): NAVETF
  - **trace** (String, 20, 선택): 추적
  - **trace_idex** (String, 20, 선택): 추적지수
  - **trace_idex_pred_pre** (String, 20, 선택): 추적지수전일대비
  - **trace_idex_pred_pre_sig** (String, 20, 선택): 추적지수전일대비기호
