# 장중투자자별매매차트요청 (ka10064)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 차트 > 장중투자자별매매차트요청(ka10064)
- **API 명**: 장중투자자별매매차트요청
- **API ID**: ka10064
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/chart`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **mrkt_tp** (String, 3, 필수): 시장구분 (000:전체, 001:코스피, 101:코스닥)
- **amt_qty_tp** (String, 1, 필수): 금액수량구분 (1:금액, 2:수량)
- **trde_tp** (String, 1, 필수): 매매구분 (0:순매수, 1:매수, 2:매도)
- **stk_cd** (String, 20, 필수): 종목코드 (거래소별 종목코드
(KRX:039490,NXT:039490_NX,SOR:039490_AL))

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **opmr_invsr_trde_chart** (LIST, , 선택): 장중투자자별매매차트
  - **tm** (String, 20, 선택): 시간
  - **frgnr_invsr** (String, 20, 선택): 외국인투자자
  - **orgn** (String, 20, 선택): 기관계
  - **invtrt** (String, 20, 선택): 투신
  - **insrnc** (String, 20, 선택): 보험
  - **bank** (String, 20, 선택): 은행
  - **penfnd_etc** (String, 20, 선택): 연기금등
  - **etc_corp** (String, 20, 선택): 기타법인
  - **natn** (String, 20, 선택): 국가
