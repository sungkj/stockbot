# 일별잔고수익률 (ka01690)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 계좌 > 일별잔고수익률(ka01690)
- **API 명**: 일별잔고수익률
- **API ID**: ka01690
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/acnt`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **qry_dt** (String, 8, 필수): 조회일자

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **dt** (String, 20, 선택): 일자
- **tot_buy_amt** (String, 20, 선택): 총 매입가
- **tot_evlt_amt** (String, 20, 선택): 총 평가금액
- **tot_evltv_prft** (String, 20, 선택): 총 평가손익
- **tot_prft_rt** (String, 20, 선택): 수익률
- **dbst_bal** (String, 20, 선택): 예수금
- **day_stk_asst** (String, 20, 선택): 추정자산
- **buy_wght** (String, 20, 선택): 현금비중
- **day_bal_rt** (LIST, , 선택): 일별잔고수익률
  - **cur_prc** (String, 20, 선택): 현재가
  - **stk_cd** (String, 20, 선택): 종목코드
  - **stk_nm** (String, 20, 선택): 종목명
  - **rmnd_qty** (String, 20, 선택): 보유 수량
  - **buy_uv** (String, 20, 선택): 매입 단가
  - **buy_wght** (String, 20, 선택): 매수비중
  - **evltv_prft** (String, 20, 선택): 평가손익
  - **prft_rt** (String, 20, 선택): 수익률
  - **evlt_amt** (String, 20, 선택): 평가금액
  - **evlt_wght** (String, 20, 선택): 평가비중
