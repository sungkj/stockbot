# ELW가격급등락요청 (ka30001)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > ELW > ELW가격급등락요청(ka30001)
- **API 명**: ELW가격급등락요청
- **API ID**: ka30001
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/elw`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **flu_tp** (String, 1, 필수): 등락구분 (1:급등, 2:급락)
- **tm_tp** (String, 1, 필수): 시간구분 (1:분전, 2:일전)
- **tm** (String, 2, 필수): 시간 (분 혹은 일입력 (예 1, 3, 5))
- **trde_qty_tp** (String, 4, 필수): 거래량구분 (0:전체, 10:만주이상, 50:5만주이상, 100:10만주이상, 300:30만주이상, 500:50만주이상, 1000:백만주이상)
- **isscomp_cd** (String, 12, 필수): 발행사코드 (전체:000000000000, 한국투자증권:3, 미래대우:5, 신영:6, NK투자증권:12, KB증권:17)
- **bsis_aset_cd** (String, 12, 필수): 기초자산코드 (전체:000000000000, KOSPI200:201, KOSDAQ150:150, 삼성전자:005930, KT:030200..)
- **rght_tp** (String, 3, 필수): 권리구분 (000:전체, 001:콜, 002:풋, 003:DC, 004:DP, 005:EX, 006:조기종료콜, 007:조기종료풋)
- **lpcd** (String, 12, 필수): LP코드 (전체:000000000000, 한국투자증권:3, 미래대우:5, 신영:6, NK투자증권:12, KB증권:17)
- **trde_end_elwskip** (String, 1, 필수): 거래종료ELW제외 (0:포함, 1:제외)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **base_pric_tm** (String, 20, 선택): 기준가시간
- **elwpric_jmpflu** (LIST, , 선택): ELW가격급등락
  - **stk_cd** (String, 20, 선택): 종목코드
  - **rank** (String, 20, 선택): 순위
  - **stk_nm** (String, 40, 선택): 종목명
  - **pre_sig** (String, 20, 선택): 대비기호
  - **pred_pre** (String, 20, 선택): 전일대비
  - **trde_end_elwbase_pric** (String, 20, 선택): 거래종료ELW기준가
  - **cur_prc** (String, 20, 선택): 현재가
  - **base_pre** (String, 20, 선택): 기준대비
  - **trde_qty** (String, 20, 선택): 거래량
  - **jmp_rt** (String, 20, 선택): 급등율
