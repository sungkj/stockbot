# 조건검색 실시간 해제 (ka10174)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 조건검색 > 조건검색 실시간 해제(ka10174)
- **API 명**: 조건검색 실시간 해제
- **API ID**: ka10174
- **Method**: POST
- **운영 도메인**: `wss://api.kiwoom.com:10000`
- **모의투자 도메인**: `wss://mockapi.kiwoom.com:10000(KRX만 지원가능)`
- **URL**: `/api/dostk/websocket`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **trnm** (String, 7, 필수): 서비스명 (CNSRCLR 고정값)
- **seq** (String, , 필수): 조건검색식 일련번호

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **return_code** (String, , 필수): 결과코드 (정상:0 나머지:에러)
- **return_msg** (String, , 필수): 결과메시지 (정상인 경우는 메시지 없음)
- **trnm** (String, , 필수): 서비스명 (CNSRCLR 고정값)
- **seq** (String, , 필수): 조건검색식 일련번호

## 4. 예시 (Examples)

### Request Example
```json
{
    "trnm": "CNSRCLR",
    "seq": "1"
}
```

### Response Example
```json
{
    'trnm': 'CNSRCLR',
    'seq' : '1',
    'return_code': 0,
    'return_msg': ''
}
```
