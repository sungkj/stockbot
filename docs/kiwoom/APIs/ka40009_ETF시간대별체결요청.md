# ETF시간대별체결요청 (ka40009)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > ETF > ETF시간대별체결요청(ka40009)
- **API 명**: ETF시간대별체결요청
- **API ID**: ka40009
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/etf`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **stk_cd** (String, 6, 필수): 종목코드

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **etfnavarray** (LIST, , 선택): ETFNAV배열
  - **nav** (String, 20, 선택): NAV
  - **navpred_pre** (String, 20, 선택): NAV전일대비
  - **navflu_rt** (String, 20, 선택): NAV등락율
  - **trace_eor_rt** (String, 20, 선택): 추적오차율
  - **dispty_rt** (String, 20, 선택): 괴리율
  - **stkcnt** (String, 20, 선택): 주식수
  - **base_pric** (String, 20, 선택): 기준가
  - **for_rmnd_qty** (String, 20, 선택): 외인보유수량
  - **repl_pric** (String, 20, 선택): 대용가
  - **conv_pric** (String, 20, 선택): 환산가격
  - **drstk** (String, 20, 선택): DR/주
  - **wonju_pric** (String, 20, 선택): 원주가격
