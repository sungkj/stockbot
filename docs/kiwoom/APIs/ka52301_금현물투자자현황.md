# 금현물투자자현황 (ka52301)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 기관/외국인 > 금현물투자자현황(ka52301)
- **API 명**: 금현물투자자현황
- **API ID**: ka52301
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/frgnistt`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **inve_trad_stat** (LIST, , 선택): 금현물투자자현황
  - **all_dfrt_trst_sell_qty** (String, 20, 선택): 투자자별 매도 수량(천)
  - **sell_qty_irds** (String, 20, 선택): 투자자별 매도 수량 증감(천)
  - **all_dfrt_trst_sell_amt** (String, 20, 선택): 투자자별 매도 금액(억)
  - **sell_amt_irds** (String, 20, 선택): 투자자별 매도 금액 증감(억)
  - **all_dfrt_trst_buy_qty** (String, 20, 선택): 투자자별 매수 수량(천)
  - **buy_qty_irds** (String, 20, 선택): 투자자별 매수 수량 증감(천)
  - **all_dfrt_trst_buy_amt** (String, 20, 선택): 투자자별 매수 금액(억)
  - **buy_amt_irds** (String, 20, 선택): 투자자별 매수 금액 증감(억)
  - **all_dfrt_trst_netprps_qty** (String, 20, 선택): 투자자별 순매수 수량(천)
  - **netprps_qty_irds** (String, 20, 선택): 투자자별 순매수 수량 증감(천)
  - **all_dfrt_trst_netprps_amt** (String, 20, 선택): 투자자별 순매수 금액(억)
  - **netprps_amt_irds** (String, 20, 선택): 투자자별 순매수 금액 증감(억)
  - **sell_uv** (String, 20, 선택): 투자자별 매도 단가
  - **buy_uv** (String, 20, 선택): 투자자별 매수 단가
  - **stk_nm** (String, 20, 선택): 투자자 구분명
  - **acc_netprps_amt** (String, 20, 선택): 누적 순매수 금액(억)
  - **acc_netprps_qty** (String, 20, 선택): 누적 순매수 수량(천)
  - **stk_cd** (String, 20, 선택): 투자자 코드
