# 프로그램매매누적추이요청 (ka90007)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 시세 > 프로그램매매누적추이요청(ka90007)
- **API 명**: 프로그램매매누적추이요청
- **API ID**: ka90007
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/mrkcond`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **date** (String, 8, 필수): 날짜 (YYYYMMDD (종료일기준 1년간 데이터만 조회가능))
- **amt_qty_tp** (String, 1, 필수): 금액수량구분 (1:금액, 2:수량)
- **mrkt_tp** (String, 5, 필수): 시장구분 (0:코스피 , 1:코스닥)
- **stex_tp** (String, 1, 필수): 거래소구분 (1:KRX, 2:NXT, 3:통합)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **prm_trde_acc_trnsn** (LIST, , 선택): 프로그램매매누적추이
  - **dt** (String, 20, 선택): 일자
  - **kospi200** (String, 20, 선택): KOSPI200
  - **basis** (String, 20, 선택): BASIS
  - **dfrt_trde_tdy** (String, 20, 선택): 차익거래당일
  - **dfrt_trde_acc** (String, 20, 선택): 차익거래누적
  - **ndiffpro_trde_tdy** (String, 20, 선택): 비차익거래당일
  - **ndiffpro_trde_acc** (String, 20, 선택): 비차익거래누적
  - **all_tdy** (String, 20, 선택): 전체당일
  - **all_acc** (String, 20, 선택): 전체누적
