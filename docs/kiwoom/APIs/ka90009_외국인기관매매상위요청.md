# 외국인기관매매상위요청 (ka90009)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 순위정보 > 외국인기관매매상위요청(ka90009)
- **API 명**: 외국인기관매매상위요청
- **API ID**: ka90009
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/rkinfo`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **mrkt_tp** (String, 3, 필수): 시장구분 (000:전체, 001:코스피, 101:코스닥)
- **amt_qty_tp** (String, 1, 필수): 금액수량구분 (1:금액(천만), 2:수량(천))
- **qry_dt_tp** (String, 1, 필수): 조회일자구분 (0:조회일자 미포함, 1:조회일자 포함)
- **date** (String, 8, 선택): 날짜 (YYYYMMDD
(연도4자리, 월 2자리, 일 2자리 형식))
- **stex_tp** (String, 1, 필수): 거래소구분 (1:KRX, 2:NXT, 3:통합)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **frgnr_orgn_trde_upper** (LIST, , 선택): 외국인기관매매상위
  - **for_netslmt_stk_cd** (String, 20, 선택): 외인순매도종목코드
  - **for_netslmt_stk_nm** (String, 20, 선택): 외인순매도종목명
  - **for_netslmt_amt** (String, 20, 선택): 외인순매도금액
  - **for_netslmt_qty** (String, 20, 선택): 외인순매도수량
  - **for_netprps_stk_cd** (String, 20, 선택): 외인순매수종목코드
  - **for_netprps_stk_nm** (String, 20, 선택): 외인순매수종목명
  - **for_netprps_amt** (String, 20, 선택): 외인순매수금액
  - **for_netprps_qty** (String, 20, 선택): 외인순매수수량
  - **orgn_netslmt_stk_cd** (String, 20, 선택): 기관순매도종목코드
  - **orgn_netslmt_stk_nm** (String, 20, 선택): 기관순매도종목명
  - **orgn_netslmt_amt** (String, 20, 선택): 기관순매도금액
  - **orgn_netslmt_qty** (String, 20, 선택): 기관순매도수량
  - **orgn_netprps_stk_cd** (String, 20, 선택): 기관순매수종목코드
  - **orgn_netprps_stk_nm** (String, 20, 선택): 기관순매수종목명
  - **orgn_netprps_amt** (String, 20, 선택): 기관순매수금액
  - **orgn_netprps_qty** (String, 20, 선택): 기관순매수수량
