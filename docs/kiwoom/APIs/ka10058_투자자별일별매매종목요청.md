# 투자자별일별매매종목요청 (ka10058)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 종목정보 > 투자자별일별매매종목요청(ka10058)
- **API 명**: 투자자별일별매매종목요청
- **API ID**: ka10058
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/stkinfo`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **strt_dt** (String, 8, 필수): 시작일자 (YYYYMMDD)
- **end_dt** (String, 8, 필수): 종료일자 (YYYYMMDD)
- **trde_tp** (String, 1, 필수): 매매구분 (순매도:1, 순매수:2)
- **mrkt_tp** (String, 3, 필수): 시장구분 (001:코스피, 101:코스닥)
- **invsr_tp** (String, 4, 필수): 투자자구분 (8000:개인, 9000:외국인, 1000:금융투자, 3000:투신, 3100:사모펀드, 5000:기타금융, 4000:은행, 2000:보험, 6000:연기금, 7000:국가, 7100:기타법인, 9999:기관계)
- **stex_tp** (String, 1, 필수): 거래소구분 (1:KRX, 2:NXT 3.통합)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **invsr_daly_trde_stk** (LIST, , 선택): 투자자별일별매매종목
  - **stk_cd** (String, 20, 선택): 종목코드
  - **stk_nm** (String, 40, 선택): 종목명
  - **netslmt_qty** (String, 20, 선택): 순매도수량
  - **netslmt_amt** (String, 20, 선택): 순매도금액
  - **prsm_avg_pric** (String, 20, 선택): 추정평균가
  - **cur_prc** (String, 20, 선택): 현재가
  - **pre_sig** (String, 20, 선택): 대비기호
  - **pred_pre** (String, 20, 선택): 전일대비
  - **avg_pric_pre** (String, 20, 선택): 평균가대비
  - **pre_rt** (String, 20, 선택): 대비율
  - **dt_trde_qty** (String, 20, 선택): 기간거래량
