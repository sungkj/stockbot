# ELW조건검색요청 (ka30005)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > ELW > ELW조건검색요청(ka30005)
- **API 명**: ELW조건검색요청
- **API ID**: ka30005
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/elw`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **isscomp_cd** (String, 12, 필수): 발행사코드 (12자리입력(전체:000000000000, 한국투자증권:000,,,3, 미래대우:000,,,5, 신영:000,,,6, NK투자증권:000,,,12, KB증권:000,,,17))
- **bsis_aset_cd** (String, 12, 필수): 기초자산코드 (전체일때만 12자리입력(전체:000000000000, KOSPI200:201, KOSDAQ150:150, 삼정전자:005930, KT:030200,,))
- **rght_tp** (String, 1, 필수): 권리구분 (0:전체, 1:콜, 2:풋, 3:DC, 4:DP, 5:EX, 6:조기종료콜, 7:조기종료풋)
- **lpcd** (String, 12, 필수): LP코드 (전체일때만 12자리입력(전체:000000000000, 한국투자증권:003, 미래대우:005, 신영:006, NK투자증권:012, KB증권:017))
- **sort_tp** (String, 1, 필수): 정렬구분 (0:정렬없음, 1:상승율순, 2:상승폭순, 3:하락율순, 4:하락폭순, 5:거래량순, 6:거래대금순, 7:잔존일순)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **elwcnd_qry** (LIST, , 선택): ELW조건검색
  - **stk_cd** (String, 20, 선택): 종목코드
  - **isscomp_nm** (String, 20, 선택): 발행사명
  - **sqnc** (String, 20, 선택): 회차
  - **base_aset_nm** (String, 20, 선택): 기초자산명
  - **rght_tp** (String, 20, 선택): 권리구분
  - **expr_dt** (String, 20, 선택): 만기일
  - **cur_prc** (String, 20, 선택): 현재가
  - **pre_tp** (String, 20, 선택): 대비구분
  - **pred_pre** (String, 20, 선택): 전일대비
  - **flu_rt** (String, 20, 선택): 등락율
  - **trde_qty** (String, 20, 선택): 거래량
  - **trde_qty_pre** (String, 20, 선택): 거래량대비
  - **trde_prica** (String, 20, 선택): 거래대금
  - **pred_trde_qty** (String, 20, 선택): 전일거래량
  - **sel_bid** (String, 20, 선택): 매도호가
  - **buy_bid** (String, 20, 선택): 매수호가
  - **prty** (String, 20, 선택): 패리티
  - **gear_rt** (String, 20, 선택): 기어링비율
  - **pl_qutr_rt** (String, 20, 선택): 손익분기율
  - **cfp** (String, 20, 선택): 자본지지점
  - **theory_pric** (String, 20, 선택): 이론가
  - **innr_vltl** (String, 20, 선택): 내재변동성
  - **delta** (String, 20, 선택): 델타
  - **lvrg** (String, 20, 선택): 레버리지
  - **exec_pric** (String, 20, 선택): 행사가격
  - **cnvt_rt** (String, 20, 선택): 전환비율
  - **lpposs_rt** (String, 20, 선택): LP보유비율
  - **pl_qutr_pt** (String, 20, 선택): 손익분기점
  - **fin_trde_dt** (String, 20, 선택): 최종거래일
  - **flo_dt** (String, 20, 선택): 상장일
  - **lpinitlast_suply_dt** (String, 20, 선택): LP초종공급일
  - **stk_nm** (String, 40, 선택): 종목명
  - **srvive_dys** (String, 20, 선택): 잔존일수
  - **dispty_rt** (String, 20, 선택): 괴리율
  - **lpmmcm_nm** (String, 20, 선택): LP회원사명
  - **lpmmcm_nm_1** (String, 20, 선택): LP회원사명1
  - **lpmmcm_nm_2** (String, 20, 선택): LP회원사명2
  - **xraymont_cntr_qty_arng_trde_tp** (String, 20, 선택): Xray순간체결량정리매매구분
  - **xraymont_cntr_qty_profa_100tp** (String, 20, 선택): Xray순간체결량증거금100구분
