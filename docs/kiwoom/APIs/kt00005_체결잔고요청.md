# 체결잔고요청 (kt00005)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 계좌 > 체결잔고요청(kt00005)
- **API 명**: 체결잔고요청
- **API ID**: kt00005
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/acnt`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **dmst_stex_tp** (String, 6, 필수): 국내거래소구분 (KRX:한국거래소,NXT:넥스트트레이드)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **entr** (String, 12, 선택): 예수금
- **entr_d1** (String, 12, 선택): 예수금D+1
- **entr_d2** (String, 12, 선택): 예수금D+2
- **pymn_alow_amt** (String, 12, 선택): 출금가능금액
- **uncl_stk_amt** (String, 12, 선택): 미수확보금
- **repl_amt** (String, 12, 선택): 대용금
- **rght_repl_amt** (String, 12, 선택): 권리대용금
- **ord_alowa** (String, 12, 선택): 주문가능현금
- **ch_uncla** (String, 12, 선택): 현금미수금
- **crd_int_npay_gold** (String, 12, 선택): 신용이자미납금
- **etc_loana** (String, 12, 선택): 기타대여금
- **nrpy_loan** (String, 12, 선택): 미상환융자금
- **profa_ch** (String, 12, 선택): 증거금현금
- **repl_profa** (String, 12, 선택): 증거금대용
- **stk_buy_tot_amt** (String, 12, 선택): 주식매수총액
- **evlt_amt_tot** (String, 12, 선택): 평가금액합계
- **tot_pl_tot** (String, 12, 선택): 총손익합계
- **tot_pl_rt** (String, 12, 선택): 총손익률
- **tot_re_buy_alowa** (String, 12, 선택): 총재매수가능금액
- **20ord_alow_amt** (String, 12, 선택): 20%주문가능금액
- **30ord_alow_amt** (String, 12, 선택): 30%주문가능금액
- **40ord_alow_amt** (String, 12, 선택): 40%주문가능금액
- **50ord_alow_amt** (String, 12, 선택): 50%주문가능금액
- **60ord_alow_amt** (String, 12, 선택): 60%주문가능금액
- **100ord_alow_amt** (String, 12, 선택): 100%주문가능금액
- **crd_loan_tot** (String, 12, 선택): 신용융자합계
- **crd_loan_ls_tot** (String, 12, 선택): 신용융자대주합계
- **crd_grnt_rt** (String, 12, 선택): 신용담보비율
- **dpst_grnt_use_amt_amt** (String, 12, 선택): 예탁담보대출금액
- **grnt_loan_amt** (String, 12, 선택): 매도담보대출금액
- **stk_cntr_remn** (LIST, , 선택): 종목별체결잔고
  - **crd_tp** (String, 2, 선택): 신용구분
  - **loan_dt** (String, 8, 선택): 대출일
  - **expr_dt** (String, 8, 선택): 만기일
  - **stk_cd** (String, 12, 선택): 종목번호
  - **stk_nm** (String, 30, 선택): 종목명
  - **setl_remn** (String, 12, 선택): 결제잔고
  - **cur_qty** (String, 12, 선택): 현재잔고
  - **cur_prc** (String, 12, 선택): 현재가
  - **buy_uv** (String, 12, 선택): 매입단가
  - **pur_amt** (String, 12, 선택): 매입금액
  - **evlt_amt** (String, 12, 선택): 평가금액
  - **evltv_prft** (String, 12, 선택): 평가손익
  - **pl_rt** (String, 12, 선택): 손익률
