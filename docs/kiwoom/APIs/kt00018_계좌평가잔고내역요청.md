# 계좌평가잔고내역요청 (kt00018)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 계좌 > 계좌평가잔고내역요청(kt00018)
- **API 명**: 계좌평가잔고내역요청
- **API ID**: kt00018
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/acnt`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **qry_tp** (String, 1, 필수): 조회구분 (1:합산, 2:개별)
- **dmst_stex_tp** (String, 6, 필수): 국내거래소구분 (KRX:한국거래소,NXT:넥스트트레이드)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **tot_pur_amt** (String, 15, 선택): 총매입금액
- **tot_evlt_amt** (String, 15, 선택): 총평가금액
- **tot_evlt_pl** (String, 15, 선택): 총평가손익금액
- **tot_prft_rt** (String, 12, 선택): 총수익률(%)
- **prsm_dpst_aset_amt** (String, 15, 선택): 추정예탁자산
- **tot_loan_amt** (String, 15, 선택): 총대출금
- **tot_crd_loan_amt** (String, 15, 선택): 총융자금액
- **tot_crd_ls_amt** (String, 15, 선택): 총대주금액
- **acnt_evlt_remn_indv_tot** (LIST, , 선택): 계좌평가잔고개별합산
  - **stk_cd** (String, 12, 선택): 종목번호
  - **stk_nm** (String, 40, 선택): 종목명
  - **evltv_prft** (String, 15, 선택): 평가손익
  - **prft_rt** (String, 12, 선택): 수익률(%)
  - **pur_pric** (String, 15, 선택): 매입가
  - **pred_close_pric** (String, 12, 선택): 전일종가
  - **rmnd_qty** (String, 15, 선택): 보유수량
  - **trde_able_qty** (String, 15, 선택): 매매가능수량
  - **cur_prc** (String, 12, 선택): 현재가
  - **pred_buyq** (String, 15, 선택): 전일매수수량
  - **pred_sellq** (String, 15, 선택): 전일매도수량
  - **tdy_buyq** (String, 15, 선택): 금일매수수량
  - **tdy_sellq** (String, 15, 선택): 금일매도수량
  - **pur_amt** (String, 15, 선택): 매입금액
  - **pur_cmsn** (String, 15, 선택): 매입수수료
  - **evlt_amt** (String, 15, 선택): 평가금액
  - **sell_cmsn** (String, 15, 선택): 평가수수료
  - **tax** (String, 15, 선택): 세금
  - **sum_cmsn** (String, 15, 선택): 수수료합 (매입수수료 + 평가수수료)
  - **poss_rt** (String, 12, 선택): 보유비중(%)
  - **crd_tp** (String, 2, 선택): 신용구분
  - **crd_tp_nm** (String, 4, 선택): 신용구분명
  - **crd_loan_dt** (String, 8, 선택): 대출일
