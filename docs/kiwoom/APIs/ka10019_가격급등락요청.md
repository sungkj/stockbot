# 가격급등락요청 (ka10019)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 종목정보 > 가격급등락요청(ka10019)
- **API 명**: 가격급등락요청
- **API ID**: ka10019
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/stkinfo`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **mrkt_tp** (String, 3, 필수): 시장구분 (000:전체, 001:코스피, 101:코스닥, 201:코스피200)
- **flu_tp** (String, 1, 필수): 등락구분 (1:급등, 2:급락)
- **tm_tp** (String, 1, 필수): 시간구분 (1:분전, 2:일전)
- **tm** (String, 2, 필수): 시간 (분 혹은 일입력)
- **trde_qty_tp** (String, 4, 필수): 거래량구분 (00000:전체조회, 00010:만주이상, 00050:5만주이상, 00100:10만주이상, 00150:15만주이상, 00200:20만주이상, 00300:30만주이상, 00500:50만주이상, 01000:백만주이상)
- **stk_cnd** (String, 1, 필수): 종목조건 (0:전체조회,1:관리종목제외, 3:우선주제외, 5:증100제외, 6:증100만보기, 7:증40만보기, 8:증30만보기)
- **crd_cnd** (String, 1, 필수): 신용조건 (0:전체조회, 1:신용융자A군, 2:신용융자B군, 3:신용융자C군, 4:신용융자D군, 7:신용융자E군, 9:신용융자전체)
- **pric_cnd** (String, 1, 필수): 가격조건 (0:전체조회, 1:1천원미만, 2:1천원~2천원, 3:2천원~3천원, 4:5천원~1만원, 5:1만원이상, 8:1천원이상)
- **updown_incls** (String, 1, 필수): 상하한포함 (0:미포함, 1:포함)
- **stex_tp** (String, 1, 필수): 거래소구분 (1:KRX, 2:NXT 3.통합)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **pric_jmpflu** (LIST, , 선택): 가격급등락
  - **stk_cd** (String, 20, 선택): 종목코드
  - **stk_cls** (String, 20, 선택): 종목분류
  - **stk_nm** (String, 40, 선택): 종목명
  - **pred_pre_sig** (String, 20, 선택): 전일대비기호
  - **pred_pre** (String, 20, 선택): 전일대비
  - **flu_rt** (String, 20, 선택): 등락률
  - **base_pric** (String, 20, 선택): 기준가
  - **cur_prc** (String, 20, 선택): 현재가
  - **base_pre** (String, 20, 선택): 기준대비
  - **trde_qty** (String, 20, 선택): 거래량
  - **jmp_rt** (String, 20, 선택): 급등률
