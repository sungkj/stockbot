# 계좌별당일현황요청 (kt00017)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 계좌 > 계좌별당일현황요청(kt00017)
- **API 명**: 계좌별당일현황요청
- **API ID**: kt00017
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/acnt`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **d2_entra** (String, 12, 선택): D+2추정예수금
- **crd_int_npay_gold** (String, 12, 선택): 신용이자미납금
- **etc_loana** (String, 12, 선택): 기타대여금
- **gnrl_stk_evlt_amt_d2** (String, 12, 선택): 일반주식평가금액D+2
- **dpst_grnt_use_amt_d2** (String, 12, 선택): 예탁담보대출금D+2
- **crd_stk_evlt_amt_d2** (String, 12, 선택): 예탁담보주식평가금액D+2
- **crd_loan_d2** (String, 12, 선택): 신용융자금D+2
- **crd_loan_evlta_d2** (String, 12, 선택): 신용융자평가금D+2
- **crd_ls_grnt_d2** (String, 12, 선택): 신용대주담보금D+2
- **crd_ls_evlta_d2** (String, 12, 선택): 신용대주평가금D+2
- **ina_amt** (String, 12, 선택): 입금금액
- **outa** (String, 12, 선택): 출금금액
- **inq_amt** (String, 12, 선택): 입고금액
- **outq_amt** (String, 12, 선택): 출고금액
- **sell_amt** (String, 12, 선택): 매도금액
- **buy_amt** (String, 12, 선택): 매수금액
- **cmsn** (String, 12, 선택): 수수료
- **tax** (String, 12, 선택): 세금
- **stk_pur_cptal_loan_amt** (String, 12, 선택): 주식매입자금대출금
- **rp_evlt_amt** (String, 12, 선택): RP평가금액
- **bd_evlt_amt** (String, 12, 선택): 채권평가금액
- **elsevlt_amt** (String, 12, 선택): ELS평가금액
- **crd_int_amt** (String, 12, 선택): 신용이자금액
- **sel_prica_grnt_loan_int_amt_amt** (String, 12, 선택): 매도대금담보대출이자금액
- **dvida_amt** (String, 12, 선택): 배당금액
