# 일별추정예탁자산현황요청 (kt00002)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 계좌 > 일별추정예탁자산현황요청(kt00002)
- **API 명**: 일별추정예탁자산현황요청
- **API ID**: kt00002
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/acnt`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **start_dt** (String, 8, 필수): 시작조회기간 (YYYYMMDD)
- **end_dt** (String, 8, 필수): 종료조회기간 (YYYYMMDD)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **daly_prsm_dpst_aset_amt_prst** (LIST, , 선택): 일별추정예탁자산현황
  - **dt** (String, 8, 선택): 일자
  - **entr** (String, 12, 선택): 예수금
  - **grnt_use_amt** (String, 12, 선택): 담보대출금
  - **crd_loan** (String, 12, 선택): 신용융자금
  - **ls_grnt** (String, 12, 선택): 대주담보금
  - **repl_amt** (String, 12, 선택): 대용금
  - **prsm_dpst_aset_amt** (String, 12, 선택): 추정예탁자산
  - **prsm_dpst_aset_amt_bncr_skip** (String, 12, 선택): 추정예탁자산수익증권제외
