# 종목시간별프로그램매매추이요청 (ka90008)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 시세 > 종목시간별프로그램매매추이요청(ka90008)
- **API 명**: 종목시간별프로그램매매추이요청
- **API ID**: ka90008
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/mrkcond`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **amt_qty_tp** (String, 1, 필수): 금액수량구분 (1:금액, 2:수량)
- **stk_cd** (String, 6, 필수): 종목코드 (거래소별 종목코드
(KRX:039490,NXT:039490_NX,SOR:039490_AL))
- **date** (String, 8, 필수): 날짜 (YYYYMMDD)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **stk_tm_prm_trde_trnsn** (LIST, , 선택): 종목시간별프로그램매매추이
  - **tm** (String, 20, 선택): 시간
  - **cur_prc** (String, 20, 선택): 현재가
  - **pre_sig** (String, 20, 선택): 대비기호
  - **pred_pre** (String, 20, 선택): 전일대비
  - **flu_rt** (String, 20, 선택): 등락율
  - **trde_qty** (String, 20, 선택): 거래량
  - **prm_sell_amt** (String, 20, 선택): 프로그램매도금액
  - **prm_buy_amt** (String, 20, 선택): 프로그램매수금액
  - **prm_netprps_amt** (String, 20, 선택): 프로그램순매수금액
  - **prm_netprps_amt_irds** (String, 20, 선택): 프로그램순매수금액증감
  - **prm_sell_qty** (String, 20, 선택): 프로그램매도수량
  - **prm_buy_qty** (String, 20, 선택): 프로그램매수수량
  - **prm_netprps_qty** (String, 20, 선택): 프로그램순매수수량
  - **prm_netprps_qty_irds** (String, 20, 선택): 프로그램순매수수량증감
  - **base_pric_tm** (String, 20, 선택): 기준가시간
  - **dbrt_trde_rpy_sum** (String, 20, 선택): 대차거래상환주수합
  - **remn_rcvord_sum** (String, 20, 선택): 잔고수주합
  - **stex_tp** (String, 20, 선택): 거래소구분 (KRX , NXT , 통합)
