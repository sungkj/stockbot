# 미체결요청 (ka10075)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 계좌 > 미체결요청(ka10075)
- **API 명**: 미체결요청
- **API ID**: ka10075
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/acnt`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **all_stk_tp** (String, 1, 필수): 전체종목구분 (0:전체, 1:종목)
- **trde_tp** (String, 1, 필수): 매매구분 (0:전체, 1:매도, 2:매수)
- **stk_cd** (String, 6, 선택): 종목코드
- **stex_tp** (String, 1, 필수): 거래소구분 (0 : 통합, 1 : KRX, 2 : NXT)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **oso** (LIST, , 선택): 미체결
  - **acnt_no** (String, 20, 선택): 계좌번호
  - **ord_no** (String, 20, 선택): 주문번호
  - **mang_empno** (String, 20, 선택): 관리사번
  - **stk_cd** (String, 20, 선택): 종목코드
  - **tsk_tp** (String, 20, 선택): 업무구분
  - **ord_stt** (String, 20, 선택): 주문상태
  - **stk_nm** (String, 40, 선택): 종목명
  - **ord_qty** (String, 20, 선택): 주문수량
  - **ord_pric** (String, 20, 선택): 주문가격
  - **oso_qty** (String, 20, 선택): 미체결수량
  - **cntr_tot_amt** (String, 20, 선택): 체결누계금액
  - **orig_ord_no** (String, 20, 선택): 원주문번호
  - **io_tp_nm** (String, 20, 선택): 주문구분
  - **trde_tp** (String, 20, 선택): 매매구분
  - **tm** (String, 20, 선택): 시간
  - **cntr_no** (String, 20, 선택): 체결번호
  - **cntr_pric** (String, 20, 선택): 체결가
  - **cntr_qty** (String, 20, 선택): 체결량
  - **cur_prc** (String, 20, 선택): 현재가
  - **sel_bid** (String, 20, 선택): 매도호가
  - **buy_bid** (String, 20, 선택): 매수호가
  - **unit_cntr_pric** (String, 20, 선택): 단위체결가
  - **unit_cntr_qty** (String, 20, 선택): 단위체결량
  - **tdy_trde_cmsn** (String, 20, 선택): 당일매매수수료
  - **tdy_trde_tax** (String, 20, 선택): 당일매매세금
  - **ind_invsr** (String, 20, 선택): 개인투자자
  - **stex_tp** (String, 20, 선택): 거래소구분 (0 : 통합, 1 : KRX, 2 : NXT)
  - **stex_tp_txt** (String, 20, 선택): 거래소구분텍스트 (통합,KRX,NXT)
  - **sor_yn** (String, 20, 선택): SOR 여부값 (Y,N)
  - **stop_pric** (String, 20, 선택): 스톱가 (스톱지정가주문 스톱가)
