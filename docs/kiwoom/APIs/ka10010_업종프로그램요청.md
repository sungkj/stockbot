# 업종프로그램요청 (ka10010)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 업종 > 업종프로그램요청(ka10010)
- **API 명**: 업종프로그램요청
- **API ID**: ka10010
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/sect`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **stk_cd** (String, 20, 필수): 종목코드 (거래소별 종목코드
(KRX:039490,NXT:039490_NX,SOR:039490_AL))

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **dfrt_trst_sell_qty** (String, 20, 선택): 차익위탁매도수량
- **dfrt_trst_sell_amt** (String, 20, 선택): 차익위탁매도금액
- **dfrt_trst_buy_qty** (String, 20, 선택): 차익위탁매수수량
- **dfrt_trst_buy_amt** (String, 20, 선택): 차익위탁매수금액
- **dfrt_trst_netprps_qty** (String, 20, 선택): 차익위탁순매수수량
- **dfrt_trst_netprps_amt** (String, 20, 선택): 차익위탁순매수금액
- **ndiffpro_trst_sell_qty** (String, 20, 선택): 비차익위탁매도수량
- **ndiffpro_trst_sell_amt** (String, 20, 선택): 비차익위탁매도금액
- **ndiffpro_trst_buy_qty** (String, 20, 선택): 비차익위탁매수수량
- **ndiffpro_trst_buy_amt** (String, 20, 선택): 비차익위탁매수금액
- **ndiffpro_trst_netprps_qty** (String, 20, 선택): 비차익위탁순매수수량
- **ndiffpro_trst_netprps_amt** (String, 20, 선택): 비차익위탁순매수금액
- **all_dfrt_trst_sell_qty** (String, 20, 선택): 전체차익위탁매도수량
- **all_dfrt_trst_sell_amt** (String, 20, 선택): 전체차익위탁매도금액
- **all_dfrt_trst_buy_qty** (String, 20, 선택): 전체차익위탁매수수량
- **all_dfrt_trst_buy_amt** (String, 20, 선택): 전체차익위탁매수금액
- **all_dfrt_trst_netprps_qty** (String, 20, 선택): 전체차익위탁순매수수량
- **all_dfrt_trst_netprps_amt** (String, 20, 선택): 전체차익위탁순매수금액
