# 신용보증금율별주문가능수량조회요청 (kt00012)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 계좌 > 신용보증금율별주문가능수량조회요청(kt00012)
- **API 명**: 신용보증금율별주문가능수량조회요청
- **API ID**: kt00012
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/acnt`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **stk_cd** (String, 12, 필수): 종목번호
- **uv** (String, 10, 선택): 매수가격

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **stk_assr_rt** (String, 1, 선택): 종목보증금율
- **stk_assr_rt_nm** (String, 4, 선택): 종목보증금율명

- **assr_30tdy_reu_amt** (String, 12, 선택): 보증금30%금일재사용금액
- **assr_40ord_alow_amt** (String, 12, 선택): 보증금40%주문가능금액
- **assr_40ord_alowq** (String, 12, 선택): 보증금40%주문가능수량
- **assr_40pred_reu_amt** (String, 12, 선택): 보증금40%전일재사용금액
- **assr_40tdy_reu_amt** (String, 12, 선택): 보증금40%금일재사용금액
- **assr_50ord_alow_amt** (String, 12, 선택): 보증금50%주문가능금액
- **assr_50ord_alowq** (String, 12, 선택): 보증금50%주문가능수량
- **assr_50pred_reu_amt** (String, 12, 선택): 보증금50%전일재사용금액
- **assr_50tdy_reu_amt** (String, 12, 선택): 보증금50%금일재사용금액
- **assr_60ord_alow_amt** (String, 12, 선택): 보증금60%주문가능금액
- **assr_60ord_alowq** (String, 12, 선택): 보증금60%주문가능수량
- **assr_60pred_reu_amt** (String, 12, 선택): 보증금60%전일재사용금액
- **assr_60tdy_reu_amt** (String, 12, 선택): 보증금60%금일재사용금액
- **entr** (String, 12, 선택): 예수금
- **repl_amt** (String, 12, 선택): 대용금
- **uncla** (String, 12, 선택): 미수금
- **ord_pos_repl** (String, 12, 선택): 주문가능대용
- **ord_alowa** (String, 12, 선택): 주문가능현금
- **out_alowa** (String, 12, 선택): 미수가능금액
- **out_pos_qty** (String, 12, 선택): 미수가능수량
- **min_amt** (String, 12, 선택): 미수불가금액
- **min_qty** (String, 12, 선택): 미수불가수량
