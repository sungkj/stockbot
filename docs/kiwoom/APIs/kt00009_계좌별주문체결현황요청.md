# 계좌별주문체결현황요청 (kt00009)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 계좌 > 계좌별주문체결현황요청(kt00009)
- **API 명**: 계좌별주문체결현황요청
- **API ID**: kt00009
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/acnt`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **ord_dt** (String, 8, 선택): 주문일자 (YYYYMMDD)
- **stk_bond_tp** (String, 1, 필수): 주식채권구분 (0:전체, 1:주식, 2:채권)
- **mrkt_tp** (String, 1, 필수): 시장구분 (0:전체, 1:코스피, 2:코스닥, 3:OTCBB, 4:ECN)
- **sell_tp** (String, 1, 필수): 매도수구분 (0:전체, 1:매도, 2:매수)
- **qry_tp** (String, 1, 필수): 조회구분 (0:전체, 1:체결)
- **stk_cd** (String, 12, 선택): 종목코드 (전문 조회할 종목코드)
- **fr_ord_no** (String, 7, 선택): 시작주문번호
- **dmst_stex_tp** (String, 6, 필수): 국내거래소구분 (%:(전체),KRX:한국거래소,NXT:넥스트트레이드,SOR:최선주문집행)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **sell_grntl_engg_amt** (String, 12, 선택): 매도약정금액
- **buy_engg_amt** (String, 12, 선택): 매수약정금액
- **engg_amt** (String, 12, 선택): 약정금액
- **acnt_ord_cntr_prst_array** (LIST, , 선택): 계좌별주문체결현황배열
  - **stk_bond_tp** (String, 1, 선택): 주식채권구분
  - **ord_no** (String, 7, 선택): 주문번호
  - **stk_cd** (String, 12, 선택): 종목번호
  - **trde_tp** (String, 15, 선택): 매매구분
  - **io_tp_nm** (String, 20, 선택): 주문유형구분
  - **ord_qty** (String, 10, 선택): 주문수량
  - **ord_uv** (String, 10, 선택): 주문단가
  - **cnfm_qty** (String, 10, 선택): 확인수량
  - **rsrv_oppo** (String, 4, 선택): 예약/반대
  - **cntr_no** (String, 7, 선택): 체결번호
  - **acpt_tp** (String, 8, 선택): 접수구분
  - **orig_ord_no** (String, 7, 선택): 원주문번호
  - **stk_nm** (String, 20, 선택): 종목명
  - **setl_tp** (String, 8, 선택): 결제구분
  - **crd_deal_tp** (String, 20, 선택): 신용거래구분
  - **cntr_qty** (String, 10, 선택): 체결수량
  - **cntr_uv** (String, 10, 선택): 체결단가
  - **comm_ord_tp** (String, 8, 선택): 통신구분
  - **mdfy_cncl_tp** (String, 12, 선택): 정정/취소구분
  - **cntr_tm** (String, 8, 선택): 체결시간
  - **dmst_stex_tp** (String, 6, 선택): 국내거래소구분
  - **cond_uv** (String, 10, 선택): 스톱가
