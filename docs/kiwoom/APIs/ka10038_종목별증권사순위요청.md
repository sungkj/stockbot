# 종목별증권사순위요청 (ka10038)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 순위정보 > 종목별증권사순위요청(ka10038)
- **API 명**: 종목별증권사순위요청
- **API ID**: ka10038
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/rkinfo`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **stk_cd** (String, 6, 필수): 종목코드 (거래소별 종목코드
(KRX:039490,NXT:039490_NX,SOR:039490_AL))
- **strt_dt** (String, 8, 선택): 시작일자 (YYYYMMDD
(연도4자리, 월 2자리, 일 2자리 형식))
- **end_dt** (String, 8, 선택): 종료일자 (YYYYMMDD
(연도4자리, 월 2자리, 일 2자리 형식))
- **qry_tp** (String, 1, 필수): 조회구분 (1:순매도순위정렬, 2:순매수순위정렬)
- **dt** (String, 2, 선택): 기간 (1:전일, 4:5일, 9:10일, 19:20일, 39:40일, 59:60일, 119:120일)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **rank_1** (String, 20, 선택): 순위1
- **rank_2** (String, 20, 선택): 순위2
- **rank_3** (String, 20, 선택): 순위3
- **prid_trde_qty** (String, 20, 선택): 기간중거래량
- **stk_sec_rank** (LIST, , 선택): 종목별증권사순위
  - **rank** (String, 20, 선택): 순위
  - **mmcm_nm** (String, 20, 선택): 회원사명
  - **buy_qty** (String, 20, 선택): 매수수량
  - **sell_qty** (String, 20, 선택): 매도수량
  - **acc_netprps_qty** (String, 20, 선택): 누적순매수수량
