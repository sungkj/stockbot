# 금현물 호가 (ka50101)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 시세 > 금현물 호가(ka50101)
- **API 명**: 금현물 호가
- **API ID**: ka50101
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/mrkcond`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **stk_cd** (String, 20, 필수): 종목코드 (M04020000 금 99.99_1kg, M04020100 미니금 99.99_100g)
- **tic_scope** (String, 2, 필수): 틱범위 (1:1틱, 3:3틱, 5:5틱, 10:10틱, 30:30틱)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **gold_bid** (LIST, , 선택): 금현물호가
  - **cntr_pric** (String, 20, 선택): 체결가
  - **pred_pre** (String, 20, 선택): 전일 대비(원)
  - **flu_rt** (String, 20, 선택): 등락율
  - **trde_qty** (String, 20, 선택): 누적 거래량
  - **acc_trde_prica** (String, 20, 선택): 누적 거래대금
  - **cntr_trde_qty** (String, 20, 선택): 거래량(체결량)
  - **tm** (String, 20, 선택): 체결시간
  - **pre_sig** (String, 20, 선택): 전일대비기호
  - **pri_sel_bid_unit** (String, 20, 선택): 매도호가
  - **pri_buy_bid_unit** (String, 20, 선택): 매수호가
  - **trde_pre** (String, 20, 선택): 전일 거래량 대비 비율
  - **trde_tern_rt** (String, , 선택): 전일 거래량 대비 순간 거래량 비율
  - **cntr_str** (String, 20, 선택): 체결강도
  - **lpmmcm_nm_1** (String, 20, 선택): K.O 접근도
  - **stex_tp** (String, 20, 선택): 거래소구분
