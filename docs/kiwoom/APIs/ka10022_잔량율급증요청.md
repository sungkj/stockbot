# 잔량율급증요청 (ka10022)

## 1. 기본 정보
- **메뉴 위치**: 국내주식 > 순위정보 > 잔량율급증요청(ka10022)
- **API 명**: 잔량율급증요청
- **API ID**: ka10022
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com(KRX만 지원가능)`
- **URL**: `/api/dostk/rkinfo`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

## 2. Request (요청)

### Header
- **api-id** (String, 10, 필수): TR명
- **authorization** (String, 1000, 필수): 접근토큰 (토큰 지정시 토큰타입("Bearer") 붙혀서 호출 
 예) Bearer Egicyx...)
- **cont-yn** (String, 1, 선택): 연속조회여부 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅)
- **next-key** (String, 50, 선택): 연속조회키 (응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅)

### Body
- **mrkt_tp** (String, 3, 필수): 시장구분 (001:코스피, 101:코스닥)
- **rt_tp** (String, 1, 필수): 비율구분 (1:매수/매도비율, 2:매도/매수비율)
- **tm_tp** (String, 2, 필수): 시간구분 (분 입력)
- **trde_qty_tp** (String, 1, 필수): 거래량구분 (5:5천주이상, 10:만주이상, 50:5만주이상, 100:10만주이상)
- **stk_cnd** (String, 1, 필수): 종목조건 (0:전체조회, 1:관리종목제외, 5:증100제외, 6:증100만보기, 7:증40만보기, 8:증30만보기, 9:증20만보기)
- **stex_tp** (String, 1, 필수): 거래소구분 (1:KRX, 2:NXT 3.통합)

## 3. Response (응답)

### Header
- **api-id** (String, 10, 필수): TR명
- **cont-yn** (String, 1, 선택): 연속조회여부 (다음 데이터가 있을시 Y값 전달)
- **next-key** (String, 50, 선택): 연속조회키 (다음 데이터가 있을시 다음 키값 전달)

### Body
- **req_rt_sdnin** (LIST, , 선택): 잔량율급증
  - **stk_cd** (String, 20, 선택): 종목코드
  - **stk_nm** (String, 40, 선택): 종목명
  - **cur_prc** (String, 20, 선택): 현재가
  - **pred_pre_sig** (String, 20, 선택): 전일대비기호
  - **pred_pre** (String, 20, 선택): 전일대비
  - **int** (String, 20, 선택): 기준률
  - **now_rt** (String, 20, 선택): 현재비율
  - **sdnin_rt** (String, 20, 선택): 급증률
  - **tot_sel_req** (String, 20, 선택): 총매도잔량
  - **tot_buy_req** (String, 20, 선택): 총매수잔량
